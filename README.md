# 세명컴고 생태전환 프로젝트(팀 프로젝트)

2학년 팀 프로젝트 (오브젝트 구성과 플레이어 움직임 구현)
---

## 프로젝트 개요

생태전환과 지속가능한 생활에 대한 교육 목적의 게임.
맵을 돌아다니며 쓰레기를 수집하며 구역을 청소하고, 수집한 쓰레기를 이용하여 장비를 제작하며 재활용과 리사이클링을 알린다.

---

## 기술 스택

1. ScriptableObject를 이용하여 자원과 장비를 관리
2. New Input System을 활용한 이동과 상호작용
3. UnitySystem 기능으로 인스펙터 창에서도 이벤트 할당 가능
4. Dictionary 구조로 오브젝트 관리를 구조화

---

## 주요 기능

1. 맵을 탐험하며 쓰레기 수집, 구역 정화
2. 수집한 쓰레기를 분해 및 전환하여 자원 획득
3. 획득한 자원으로 탐험에 사용하는 장비 제작
4. 업그레이드한 장비로 엔딩 스테이지까지 진행
